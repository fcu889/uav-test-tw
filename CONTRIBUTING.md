# Contributing

感謝你協助改善全國無人機測驗中心。

## 可協助項目

- 回報題庫文字、答案或分類錯誤
- 回報官方題庫或法規公告更新
- 改善手機版閱讀與測驗體驗
- 改善搜尋引擎可讀的靜態題庫頁
- 補充安全飛行、空域判斷、考照準備等教學文章

## 回報題庫問題

請提供：

- 題庫類型：普通操作證、專業操作證或屆期換證
- 題號或頁面 URL
- 目前網站顯示內容
- 你認為應修正的內容
- 可查證來源，例如民航局公告、官方題庫或法規連結

## Pull Request 流程

1. 建立清楚的分支名稱。
2. 題庫更新、UI 調整與文件更新請盡量分開。
3. 執行 `npm run validate`。
4. 在 PR 說明中列出資料來源、驗證方式與影響頁面。
