# Security Policy

## Supported Versions

目前維護主分支與公開網站：

https://uav-test-tw.netlify.app

## Reporting a Vulnerability

本專案是純靜態網站，不提供使用者帳號登入，也不處理付款資料。若你發現下列問題，請透過 GitHub issue 回報：

- XSS 或 HTML injection
- 第三方腳本或 CDN 風險
- 錯誤的外部連結或假冒官方來源
- 部署設定、sitemap、robots 或 canonical 設定錯誤

請在回報中包含重現步驟、影響範圍與建議修正方式。若問題可被直接濫用，請先提供縮短後的重現方式，不要公開完整攻擊 payload。
